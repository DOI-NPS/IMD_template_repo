# Conributing: This file describes how people can contribute to the repository.
This is a template repository containing just the bare essential files and content for a DGEC compliant repository. Changes that improve the utility of this repository are encouraged.

## Contributing Requirements: 
Make sure your Pull Request is net-neutural for test coverage: PRs that decrease test coverage will not be merged. As current test coverage is 0%, this should be a low bar.
