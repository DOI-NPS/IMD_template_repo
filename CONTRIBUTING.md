# Conributing: This file describes how people can contribute to the repository.

This is a template repository containing just the bare essential files and content for a DGEC compliant repository. Additions and contributions are not encouraged as they likely would involve deviating from the goal of the repository: to be a minimally compliant repository that others can use to make sure their own repositories are also compliant.

Contributing Requirements: Make sure your Pull Request is net-neutural for test coverage: PRs that decrease test coverage will not be merged. As current test coverage is 0%, this should be a low bar. PRs that decrease test coverage will not be merged.
